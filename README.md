Paradise Nursery adalah aplikasi web e-commerce sederhana yang menjual tanaman hias.
Aplikasi ini dibuat menggunakan React dan Redux Toolkit.

## Fitur Utama
- Halaman utama dengan nama perusahaan dan tombol "Mulai"
- Daftar tanaman berdasarkan kategori
- Keranjang belanja dengan Redux
- Tambah, hapus, dan ubah jumlah produk
- Navigasi antar halaman

## Teknologi
- React
- Redux Toolkit
- React Router
- CSS
